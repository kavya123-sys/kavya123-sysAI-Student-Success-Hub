# kavya123-sysAI-Student-Success-Hub
AI Student Success Hub 🚀

An AI-powered student career development platform designed to help diploma and engineering students plan their academic and professional journey. The platform provides career analysis, skill assessment, semester roadmaps, project recommendations, and career readiness tracking through an interactive and modern dashboard.
